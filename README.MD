## Clone Repository

```bash

```

```bash
cd FoodExpress-FullStack-Food-Delivery-App
```

---

# Backend Setup

```bash
cd Backend
```

Install dependencies

```bash
npm install
```

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

RAZORPAY_KEY_ID=your_key_id

RAZORPAY_KEY_SECRET=your_key_secret
```

Run Backend

```bash
npm start
```

or

```bash
npm run dev
```
